# Basic Chatbot

## Description

Basic Chatbot is a GUI-based chatbot developed using Python and Tkinter. The chatbot responds to user queries using predefined rules and conditional statements. It provides information about Python, internship projects, date and time, and other common questions through a simple and interactive graphical interface.

This project was developed as part of the CodeAlpha Internship Program to demonstrate Python programming concepts, GUI development, event handling, conditional logic, and file handling.

## Features

* Interactive graphical user interface (GUI)
* Greeting and conversation support
* Date and time responses
* Information about Python
* Internship project information
* Help menu with available commands
* Save chat history as a TXT file
* Clear chat functionality
* Input validation and error handling
* User-friendly Tkinter interface

## Technologies Used

* Python 3
* Tkinter
* Datetime Module
* File Handling

## Concepts Demonstrated

* If-Else Conditions
* Functions
* Classes and Objects
* Event-Driven Programming
* Input and Output
* File Handling
* GUI Development with Tkinter

## How to Run

1. Download or clone the repository.
2. Open a terminal in the project folder.
3. Run the following command:

bash
python chatbot.py
